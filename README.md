# 教員残業代シミュレーター

公立学校（小中高）の教員がどれだけ搾取されているかを可視化するツールです。

## 📊 このツールについて

月額基本給と時間外在校等時間を入力すると、**労働基準法に基づいて本来払われるべき割増賃金**と**実際に支給されている額（教職調整額＋部活動手当）**を比較し、未払い相当額（搾取額）を試算できます。

### 給特法（給与特別措置法）について
給特法により、公立学校教員には残業代が支払われません。代わりに月額基本給の4～10%の「教職調整額」のみが支給されています。このツールは、その差額がいかに大きいかを明らかにするために開発されました。

## 📝 入力項目

- **月額基本給** - 基本給のみ（手当込み額面ではなく）
- **教職調整額の支給率** - 年度ごとの支給率を選択（2026年1月から5%）
- **部活動手当（月額）** - 部活動指導手当（月額ベース）
- **平日・土曜・日曜の時間外在校等時間** - スライダーで調整

## 📊 計算結果

- **基礎賃金** - 月額基本給を所定労働時間で割った単価
- **実質時給** - 総収入を実働時間で割った単価
- **残業1時間あたりの実質支給額** - 教職調整額＋部活動手当を残業時間で割った単価
- **搾取額（月額）** - 本来払われるべき割増賃金との差額
- **搾取額（年間換算）** - 月額の搾取額を12倍したもの

## 🔗 計算ロジック

### 所定労働時間
週38時間45分（1日7時間45分×5日）を基準とします

### 割増賃金の計算
1. **平日＋土曜（所定休日）を合算**
   - 月60時間まで：基礎賃金 × 1.25倍
   - 超過分：基礎賃金 × 1.5倍

2. **日曜（法定休日）**
   - 常に基礎賃金 × 1.35倍

### 搾取額
= 本来払われるべき割増賃金 - （教職調整額 + 部活動手当）

## 💾 シェア機能

計算結果を画像カード（OGP風）として生成し、以下の方法でシェアできます：

- **X（旧Twitter）** - 共有ボタンから投稿画面へ遷移
- **その他のSNS** - 画像を保存して手動で投稿
- **URLで共有** - 入力条件を含むURLをコピー＆ペースト
