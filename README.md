# 給特法による搾取額計算ツール

日本の公立学校教員が「給特法（定額働かせ放題）」によってどれだけ搾取されているかを計算するツールです。

## 📊 使い方

1. [こちら](https://EDUCANS.github.io/Unlimited-Work-for-a-Flat-Fee/teacher_overtime_calculator.html)にアクセス
2. 以下の情報を入力してください：
   - 基本給
   - 月の残業時間
   - その他の勤務情報
3. 計算ボタンを押して結果を確認

## 📝 機能

- 残業代の計算
- 給特法の影響を数値化
- わかりやすいグラフ表示

## ⚠️ 注意

このツールは教育問題への認識を高めるためのものです。正確な給与計算については、所属の教育委員会にお問い合わせください。
